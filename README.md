# ZEN OF ZIG  
**UPDATED WITH EVERY STABLE RELEASE**

## Programming Book Series

Zen of Zig is a book series that teaches the Zig programming language from the ground up. It's a personal journey by the author, who walks through the fundamentals and gradually moves into more advanced techniques, volume by volume. Each chapter in the first volume includes playgrounds for experimenting with the code directly in the browser. All books are kept up to date with the latest stable version of Zig.
